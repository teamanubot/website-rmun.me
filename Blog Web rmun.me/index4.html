Tutorial Install Manguni Squad

1. Download Manguni Squad Disini
2. Download Password ZIP Disini
3. Ekstrak ManguniSquad_44 Totalitas.zip (Membutuhkan Waktu Agak Lama)
4. Buka Folder ManguniSquad_44 Totalitas - By TeamAnuBot
5. Jangan Install ManguniSquad_44 Totalitas.apk Dulu Tetapi Buka Folder obb
6. Pindahkan/Copy Folder com.BigDade.ManguniSquad Ke android > obb
7. Buka Folder data Pindahkan/Copy Folder com.BigDade.ManguniSquad Ke android > data
8. Install ManguniSquad_44 Totalitas.apk
